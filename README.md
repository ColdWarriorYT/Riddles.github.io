# Riddle Site

Welcome to the Riddle Site project! This project is designed to provide a collection of riddles, puzzles, and brain teasers for users to enjoy. Below you will find information on how to set up and use the project.

## Usage

- Explore the riddles available on the site.
- Challenge yourself or others to solve the riddles.
- Feel free to add your own riddles by modifying the `index.html` file.

## Contributing

Contributions are welcome! If you have suggestions for improvements or additional riddles, please feel free to submit a pull request.
