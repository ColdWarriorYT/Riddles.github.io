# Riddle Site

Welcome to the Riddle Site project! This project is designed to provide a collection of riddles, puzzles, and brain teasers for users to enjoy. Below you will find information on how to set up and use the project.

## Project Structure

The project consists of the following files and directories:

```
riddle-site
├── index.html        # Main entry point of the riddle site
├── css
│   └── styles.css    # Styles for the riddle site
├── js
│   └── main.js       # JavaScript functionality (currently empty)
├── .gitignore        # Files and directories to be ignored by Git
└── README.md         # Documentation for the project
```

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd riddle-site
   ```

2. **Open the Project**
   Open `index.html` in your preferred web browser to view the riddle site.

3. **Styles**
   The site uses an external CSS file located in the `css` directory. You can modify `styles.css` to change the appearance of the site.

4. **JavaScript Functionality**
   The `js/main.js` file is available for any future JavaScript enhancements. You can add scripts to improve interactivity or functionality.

## Usage

- Explore the riddles available on the site.
- Challenge yourself or others to solve the riddles.
- Feel free to add your own riddles by modifying the `index.html` file.
